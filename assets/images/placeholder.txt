## Image Guidelines

### Format
- Use WebP format for better compression
- Provide JPEG fallbacks for older browsers
- Optimize images for web (under 500KB each)

### Quality
- High resolution for retina displays
- Compressed for fast loading
- Consistent aspect ratios

### Accessibility
- Include alt text for all images
- Use descriptive filenames
- Ensure sufficient color contrast

## Optimization Tools

### Online Tools
- TinyPNG.com - Image compression
- Squoosh.app - Format conversion
- ImageOptim - Batch optimization

### Command Line
```bash
# Install imagemin-cli
npm install -g imagemin-cli

# Optimize images
imagemin assets/images/*.jpg --out-dir=assets/images/optimized
```
